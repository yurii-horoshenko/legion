What it does

Everything in one place, nothing in the cloud.

⊞
Agent Catalog

174+ agents from the agency-agents registry. Filter by category, search by name, add to any project in one click.

⚙
Multi-provider Models

Configure Anthropic, OpenAI, Google, Mistral, or Ollama. Keys stored locally, never in the cloud. Load available models directly from each API.

◎
Per-agent Workspaces

Every agent gets its own folder — tasks, memories, cron jobs, channels, and identity files, all in plain Markdown and JSON.

⚡
Zero dependencies

The server runs on Node.js stdlib — no Express, no database, no Docker. One file, one command, works on macOS and Linux.

How it works

Three steps from zero
to running agents.

01 — Setup
Connect providers

Add your API keys for Anthropic, OpenAI, or point to a local Ollama instance. Load available models automatically.

02 — Configure
Create a project

Name it, link a folder on disk. Legion creates a .legion/ workspace — version-controllable, human-readable.

03 — Manage
Add agents

Pick from the catalog. Each agent gets identity files, task boards, memory logs, and cron scheduling — ready to edit.
